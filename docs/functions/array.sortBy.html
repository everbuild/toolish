    Function sortBy

    • sortBy<T>(array: T[], key: keyof T, order?: number): T[]

      Sorts an array in place by a specific key.

      If you need to preserve the original array, make a copy first with e.g. slice.

      Values of differing types are sorted in groups in this order: nullish, booleans, numbers (including numeric strings), strings (case-insensitive), symbols, objects (by their JSON representation), everything else.

      NOTE this is unlike default Array.sort behaviour, which sorts by string representation and moves all undefined elements to the end.
